Dose Tracker Application

The Dose Tracker application is a mobile Android solution that supports users' ongoing efforts towards medication adherence with features to manage and track medication schedules. The application offers efficient, reliable CRUD functionality for medication records, a local, persistent database for all data retention, and user-friendly reminder notifications to help encourage satisfactory medication compliance.

Installation Instructions

  • Download the APK file
  • Enable "Install from Unknown Sources" on Android device
  • Open APK and install
  • Launch Dose Tracker app

Features

  • Add, edit, delete medication records
  • Local Room/SQLite database storage
  • Medication reminder notifications
  • User-friendly interface
